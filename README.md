# Phone-Review-Website

This repository is a phone review website created fot django class lab session

## Installation

Install Django
```bash
pip3 install Django
```

Run application
```bash
python3 manage.py runserver
```

Download bootstrap if needed
```bash
pip3 install django-bootstrap-v5
```
